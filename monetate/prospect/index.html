Monetate Prospect

Prospect is a browser extension that displays the testing, product recommendation and analytics tools used by a website. It also shows the linkedin contacts, jigsaw contacts and traffic information from compete.com

Install Chrome Extension Install Safari Extension

Installation instructions for Chrome

With the recent versions of Chrome, for security reasons Google has decided to allow users to install third party extensions only from the Chrome web store. Unfortunately we cannot put this extension on the web store since we don't want anyone outside Monetate to get it. Please follow these instructions to install this extension. If there are any difficulties, send me an email.

STEP 1: Download the extension (chrome.crx) by clicking the button above. It will generally get saved in your Downloads folder. You can also right click and 'save as' in your preferred location.

STEP 2: Click on Window -> Extensions

STEP 3: Drag and drop the downloaded extension (chrome.crx) on the extensions page

2. Is there a Firefox version?

No, this extension is currently available only for Chrome and Safari. If you need a Firefox version let me know.

3. Is this a ghostery replacement?

This extension eliminates all the unwanted information from Ghostery and also shows a bunch of other useful things. Prospect can definitely be used as a Ghostery replacement.
